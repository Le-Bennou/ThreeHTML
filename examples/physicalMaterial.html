Uses of Physical Material Map's

The physical material needs multiple texture to mimic the "real" comportement of litght when boucne on an object

Color Map

the albedo (or difffuse) map is the colors of the object

three-material{
            --map:url('paht/to/image');
}

Normal Map

This map add some shadows to creat the illusion of bumpiness on the object
the "--normalscale" property let you to exagerate or smooth the effect (default is 1) 

three-material{
            --normalmap:url('paht/to/image');
            --normalscale:1;
}

AO Map

This map add some shadows on obstructed area of the object
the "--aomapintensity" property let you to exagerate or smooth the effect (default is 1) 

three-material{
            --aomap:url('paht/to/image');
            --aomapintensity:1;
}
