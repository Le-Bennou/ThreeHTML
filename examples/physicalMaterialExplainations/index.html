The maps

Color Map

the albedo (or difffuse) map is the colors of the object

three-material{
            --map:url('paht/to/image');
}

Normal Map

This map add some shadows to creat the illusion of bumpiness on the object
the "--normalscale" property let you to exagerate or smooth the effect (default is 1) 

three-material{
            --normalmap:url('paht/to/image');
            --normalscale:1;
}

AO Map

This map add some shadows on obstructed area of the object
the "--aomapintensity" property let you to exagerate or smooth the effect (default is 1) 

three-material{
            --aomap:url('paht/to/image');
            --aomapintensity:1;
}

Roughness Map

A grayscale image, black => rought, white => shiny

three-material{
            --roughnessmap:url('paht/to/image');
            
}

Displacement Map

A grayscale image, black => down, white => up

Since it displace the vertices of the mesh it needs a detailed model

the "--displacementscale" property let you to exagerate or smooth the effect (default is 1)

three-material{
        --displacementmap:url('paht/to/image');
        --displacementscale: 0.1;
        
}
